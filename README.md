# StudySync AI

StudySync AI is a React-based study companion that helps students plan tasks, track topic mastery, save revision notes, and receive smart study recommendations.

## Problem Statement

Students often study without a clear plan, forget important deadlines, and struggle to identify weak subjects. StudySync AI solves this by providing one personalized dashboard for planning, progress tracking, and revision support.

## Features

- User authentication with protected routes
- Dashboard with task and subject insights
- Study planner with CRUD operations
- Subject mastery tracking
- Revision notes with CRUD operations
- Persistent local storage fallback
- Firebase-ready service layer for real backend integration

## Tech Stack

- React
- Vite
- React Router
- Firebase
- Recharts
- CSS

## Setup

1. Install dependencies with `npm install`
2. Copy `.env.example` to `.env`
3. Add Firebase credentials if you want live backend services
4. Run `npm run dev`
